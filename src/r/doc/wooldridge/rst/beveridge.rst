beveridge
---------

Description
~~~~~~~~~~~

Data loads lazily. Type data(beveridge) into the console.

Usage
~~~~~

::

    data(beveridge)

Format
~~~~~~

A data.frame with 135 rows and 8 variables:

-  month. dec 200 through feb 2012

-  urate. unemployment rate, percent

-  vrate. vacancy rate, percent

-  t. linear time trend

-  urate\_1. L.urate

-  vrate\_1. L.vrate

-  curate. D.urate

-  cvrate. D.vrate

~~~~~~

~~~~~~~~
