campus
------

Description
~~~~~~~~~~~

Data loads lazily. Type data(campus) into the console.

Usage
~~~~~

::

    data(campus)

Format
~~~~~~

A data.frame with 97 rows and 7 variables:

-  enroll. total enrollment

-  priv. =1 if private college

-  police. employed officers

-  crime. total campus crimes

-  lcrime. log(crime)

-  lenroll. log(enroll)

-  lpolice. log(police)
