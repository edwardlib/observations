audit
-----

Description
~~~~~~~~~~~

Data loads lazily. Type data(audit) into the console.

Usage
~~~~~

::

    data(audit)

Format
~~~~~~

A data.frame with 241 rows and 3 variables:

-  w. =1 if white app. got job offer

-  b. =1 if black app. got job offer

-  y. b - w
