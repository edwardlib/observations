Contrived Collinear Data
------------------------

Description
~~~~~~~~~~~

The ``Mandel`` data frame has 8 rows and 3 columns.

Usage
~~~~~

::

    Mandel

Format
~~~~~~

This data frame contains the following columns:

x1
    first predictor.

x2
    second predictor.

y
    response.
