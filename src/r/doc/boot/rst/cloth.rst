Number of Flaws in Cloth
------------------------

Description
~~~~~~~~~~~

The ``cloth`` data frame has 32 rows and 2 columns.

Usage
~~~~~

::

    cloth

Format
~~~~~~

This data frame contains the following columns:

``x``
    The length of the roll of cloth.

``y``
    The number of flaws found in the roll.
