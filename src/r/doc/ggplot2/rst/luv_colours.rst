``colors()`` in Luv space
-------------------------

Description
~~~~~~~~~~~

All built-in ``colors()`` translated into Luv colour space.

Usage
~~~~~

::

    luv_colours

Format
~~~~~~

A data frame with 657 observations and 4 variables:

L,u,v
    Position in Luv colour space

col
    Colour name
