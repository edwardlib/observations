Simulation Data for Ecological Inference
----------------------------------------

Description
~~~~~~~~~~~

This dataframe contains a simulated data set to illustrate the models
for ecological inference.

Usage
~~~~~

::

    data(eidat)

Format
~~~~~~

A table containing 4 variables ("t0", "t1", "x0", "x1") and 10
observations.
