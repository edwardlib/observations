A case-control data involving four SNPs with missing genotype
-------------------------------------------------------------

Description
~~~~~~~~~~~

This is a simulated data of four SNPs with their alleles coded in
characters. The variable y contains phenotypes (1=case, 0=control).

Usage
~~~~~

::

    data(fsnps)

Format
~~~~~~

A data frame
