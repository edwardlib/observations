The River Nidd Data
-------------------

Description
~~~~~~~~~~~

These data represent annual maximal levels of the River Nidd in
Yorkshire. These data are suitable for analysis with ``gev``.

Usage
~~~~~

::

    data(nidd.annual)

Format
~~~~~~

A numeric vector containing 35 observations.
