The River Nidd Data
-------------------

Description
~~~~~~~~~~~

These data represent high river levels of the River Nidd in Yorkshire
above a threshold value of 65. These data are suitable for analysis with
``gpd``.

Usage
~~~~~

::

    data(nidd.thresh)

Format
~~~~~~

A numeric vector containing 154 observations.
