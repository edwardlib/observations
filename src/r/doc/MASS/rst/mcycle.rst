Data from a Simulated Motorcycle Accident
-----------------------------------------

Description
~~~~~~~~~~~

A data frame giving a series of measurements of head acceleration in a
simulated motorcycle accident, used to test crash helmets.

Usage
~~~~~

::

    mcycle

Format
~~~~~~

``times``
    in milliseconds after impact.

``accel``
    in g.
